# movieApp

## Requisites

CocoaPods: https://guides.cocoapods.org/using/getting-started.html

XCode 11

## Installation

`cd /movieApp`

`pod install`

`open movieApp.xcworkspace/`

Open with Xcode ./movieApp.xcworkspace
